# sandilescwebu.github.io

Personal portfolio website for Sandile Scwebu — final-year BSc Information Technology 
student at North-West University.

## Live Site
https://sandilescwebu.github.io

## Sections
- **About Me** — Bio, professional photo, career goals
- **Resume** — Interactive CV with download option
- **Projects** — Fund Distribution System, Library Management System, 
  Network Infrastructure Design, Operational Database System
- **Skills** — Programming languages, databases, tools, soft skills
- **Contact** — Email, phone, LinkedIn, GitHub, contact form

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## Features
- Responsive design (mobile, tablet, desktop)
- Dark / light mode toggle
- Smooth scroll navigation
- Scroll-triggered animations
- Interactive project cards

## Setup
1. Clone the repository
2. Open `index.html` in your browser
3. Or visit the live site above
